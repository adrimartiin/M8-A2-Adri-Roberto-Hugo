GIT Log:

  • El comando git log sirve para ver los cambios realizados en el proyecto.

    • El comando git log muestra los cambios realizados en el proyecto. Sus parámetros más comunes son: --oneline, --graph, --decorate y --all.

    Los parámetros del comando git log son:

    • --oneline: Muestra cada confirmación en una sola línea.
    • --graph: Muestra un gráfico de los commits y ramas.
    • --decorate: Muestra las etiquetas y ramas en las que se encuentra cada commit.
    • --all: Muestra todos los commits, incluyendo los que están en ramas no fusionadas.
